# 🎨 Marlyn Theme

### Inspirado en <https://github.com/Gentleman-Programming/gentleman-kanagawa-blur>

## 📦 Installation

### [lazy.nvim](https://github.com/folke/lazy.nvim) (Recommended)

```lua
{
  "ikaros/Marlyn-theme",
  lazy = false,
  priority = 1000,
  config = function()
    require("marlyn").setup({
      variant = "pastel", -- Theme variant: "pastel" (default), "vibrant", or "aurora"
    })
  end,
}
```
