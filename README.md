# 🎨 Marlyn Theme

## 📦 Installation

### [lazy.nvim](https://github.com/folke/lazy.nvim) (Recommended)

```lua
{
  "ikaros/Marlyn-theme",
  lazy = false,
  priority = 1000,
  config = function()
    require("marlyn").setup({
      -- Optional configuration here
    })
    vim.cmd("colorscheme marlyn")
  end,
}
```
